# Custom Educational OS

A custom-built educational operating system kernel developed from scratch using **C and x86 Assembly**, designed to run in a **bare-metal freestanding environment**. Tested and run on **QEMU**, this OS provides a simple **command-line interface (CLI)** with a basic **authentication layer**.

## Features

- Custom Bootloader initializes the system
- Kernel written in C and Assembly
- Command-Line Interface (CLI) for user interaction
- Basic commands: `ls`, `clear`, `reboot`, `status`, `help`
- Simple authentication using a master key
- Educational platform for learning OS development
